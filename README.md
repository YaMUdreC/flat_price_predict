# flat_price_predict

В данном проекте решается задача **регрессии** — предсказание стоимости квартиры на основе её характеристик (площадь, район, этаж, коммуникации и др.).

Проект включает полный цикл Data Science:
- разведочный анализ данных (EDA)
- очистка и проверка качества данных
- feature engineering (создание новых признаков)
- обучение и сравнение моделей
- оценка качества по метрике **RMSE**
- генерация предсказаний для test

---

##  Данные
Используются следующие файлы:
- `data.csv` — обучающая выборка (~100k объектов)
- `test.csv` — тестовая выборка (без target)
- `solution_example_full.csv` — пример submission

**Целевая переменная:** `price`
